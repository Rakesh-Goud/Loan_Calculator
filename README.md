A simple desktop application built using **Python** and **Tkinter** to calculate **monthly** and **total loan payments** based on the loan amount, annual interest rate, and loan duration.

---

## Features
- Interactive and user-friendly GUI using Tkinter  
- Calculates:
  - Monthly Payment  
  - Total Payment (over the full loan period)
- Takes user input for loan amount, interest rate, and years  
- Implements Object-Oriented Programming (OOP) principles in Python  

---

## How It Works
1. Enter the following details:
   - **Annual Interest Rate** (e.g., 7.5)
   - **Number of Years** (e.g., 10)
   - **Loan Amount** (e.g., 100000)
2. Click on **Compute Payment**  
3. The program displays:
   - **Monthly Payment**
   - **Total Payment**
